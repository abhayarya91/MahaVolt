## Features
- 🔍 Real-time password strength & entropy analysis
- ☠️ HIBP breach checking (k-Anonymity — password never transmitted)
- ⚡ Attack simulation (Brute Force / Dictionary / GPU / Rainbow Table)
- 🔐 Cryptographic password & passphrase generator
- ⚑ NIST SP 800-63B compliance checker
- ◈ SOC-style dashboard with JSON/CSV export
- 🔒 100% local processing — zero data storage
